# Referent

Минимальное приложение на Next.js с TypeScript.

## Установка

Установите зависимости:

**С помощью npm:**
```powershell
npm install
```

**С помощью pnpm:**
```powershell
pnpm install
```

## Запуск

Запустите сервер разработки:

**С помощью npm:**
```powershell
npm run dev
```

**С помощью pnpm:**
```powershell
pnpm dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка

Для сборки production версии:

**С помощью npm:**
```powershell
npm run build
```

**С помощью pnpm:**
```powershell
pnpm build
```

Для запуска production сервера:

**С помощью npm:**
```powershell
npm start
```

**С помощью pnpm:**
```powershell
pnpm start
```
